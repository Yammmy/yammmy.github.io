How to create a new blog using hexo?

Published on: Apr 2, 2017
Tags: hexo cloud9 github-page

写在前面：

本次我们用来创建blog的工具是cloud9、hexo和github page

  • cloud9 是一个网页代码编辑器，详情可以点击下面的链接:
    Cloud9 IDE
  • hexo是一个博客框架: Hexo
  • github page:
    GitHub Pages

进入正题：

  1. use cloud9 to create a workspace

  • 注册cloud9: https://c9.io/signup
    注意：注册需要信用卡

  • create a workspace using html5 in cloud9

  1. install hexo using npm

使用 Cloud 9 開發的話，Node.js 已經有預設安裝了，所以直接安裝 Hexo 即可：
$ npm install hexo-cli -g

接著，我們用 hexo init 來初始化一個部落格工作環境：
$ hexo init blog

切换到blog並安裝環境依賴及需求：
$ cd blog
$ npm install

接着可以像rails一样run server来本地打开网址预览和测试：
$ hexo s -p 8080

这样点击网址就可以本地测试了： http://localhost:8080/

  1. 用hexo实作blog

其實整體流程很簡單：創建一篇新文章 (new) → 編輯文章 → 產生靜態檔 (generate) → 佈署到 GitHub (deploy)

  • create a new blog

$ hexo new [layout] your-article-title

這邊生出來的文章檔案會在 source/_drafts/、source/_posts/ 或 source/ 下，單看你選擇什麼 layout 的模板。

Hexo 有三種預設 layout: draft(草稿)、post(公開文章) 和page(頁面)，下指令時沒填 layout 預設是 post。
你也可以在 scaffolds 中編輯或添加你要的模板！

  • edit the blog
    使用markdown格式编辑blog

  • 如何在文章中插入圖片？

我們可以先在 blog/source/ 下建立 img 資料夾，將你要使用的圖檔放在這資料夾中後，就可以在文章中引用：

  • deploy to github
    正式部署之前，需要做3件事

a. 在github上创建一个repo
在你的 GitHub 開一個名為 your-github-username.github.io.git 的 repo
username一定要和你的github id一样 大小写无所谓

b. 设置config

注：当然你也可以不用master分支

c. install git deployer to deploy it to Github
$ npm install hexo-deployer-git --save

设置好之后，就可以部署了：
$ hexo deploy

ok, 现在点击你的github链接就可以看到部署上去的内容了：
https://github.com/your-github-username/your-github-username.github.io.git

為了方便起見，generate 和 deploy 兩道指令都提供了選項，讓你可以 產生靜態檔後就佈署 或 佈署前產生靜態檔：

  1. change theme

预设的thema是landscape
如果觉得不好看，可以换一个

Hexo 有很多好看的主題可以選擇，請先到 Hexo Theme 參觀並選擇你要的主題，然後就可以輕鬆的作更換囉！
更換主題流程很簡單：複製你要的主題 (git clone) → 編輯設定 (_config.yml)

複製你要的主題 (git clone)
以我為例，我選用得主題是 nexT：

編輯設定 (_config.yml)，替换landscape

注意：
每次更新文件之后，都需要重新generate 和 deploy

  1. Finally

点击链接：https://your-github-username.github.io
这样就成功新建了你的新blog了。
